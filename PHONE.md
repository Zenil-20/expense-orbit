# Install Expense Orbit as a real app on your phone

No cable, no Play Store, no Android SDK, no cost. Works over Wi-Fi.

## The short version

```
npm run phone-app
```

A QR code and an HTTPS URL appear in your terminal. Open that URL on your phone → tap **Install app**. The app appears in your Android app drawer (or iOS home screen) as a real installed application, opens standalone with no browser UI, and hot-reloads whenever you save code on your laptop.

## One-time setup (90 seconds)

```
npm install
npm --prefix client install
```

That's it. No Android Studio, no JDK, no cable.

## Every session

```
npm run phone-app
```

What this does, in one command:

1. Starts the backend (`localhost:5000`).
2. Starts Vite dev server (`localhost:5173`).
3. Opens a free HTTPS tunnel via `localtunnel` — something like `https://giant-sloth-42.loca.lt`.
4. Prints a QR code you scan with your phone camera.

### Installing on Android (Chrome)

1. Scan the QR → opens in Chrome.
2. A one-time `loca.lt` notice asks you to tap **Continue**. Do it.
3. Chrome menu (⋮) → **Install app**. (On modern Android, this creates a real WebAPK — it shows up in your app drawer, in Settings → Apps, separate task in the switcher. Android treats it the same as a Play Store app.)
4. Launch from the home screen / app drawer. No URL bar, no browser chrome.

### Installing on iOS (Safari)

1. Scan the QR → opens in Safari.
2. Share → **Add to Home Screen** → Add.
3. Icon on home screen launches the app standalone.

## Editing code after install

- Change any `.jsx` / `.css` → the **installed app on your phone reloads automatically** within ~200ms (Vite HMR through the tunnel).
- Change backend code → nodemon restarts the API; pull-to-refresh on the phone.

No rebuild, no re-install, no redeploy. Edit on laptop, see it on phone instantly.

## If you don't need the tunnel (same Wi-Fi only)

If you just want to use the app while on the same Wi-Fi as your laptop and don't care about Chrome's install prompt, skip the tunnel:

```
npm run phone
```

That gives you `http://<laptop-LAN-IP>:5173`. Open on phone → browser menu → **Add to Home Screen**. On Chrome 94+ this still installs as a WebAPK from LAN HTTP as long as the PWA manifest is valid (ours is).

Requires a one-time Windows firewall rule so the phone can reach port 5173:

```powershell
New-NetFirewallRule -DisplayName "Vite 5173" -Direction Inbound -Protocol TCP -LocalPort 5173 -Action Allow
New-NetFirewallRule -DisplayName "Express 5000" -Direction Inbound -Protocol TCP -LocalPort 5000 -Action Allow
```

## Troubleshooting

- **"Install app" option not shown in Chrome** → use `npm run phone-app` (HTTPS tunnel). Chrome reserves the native install UI for HTTPS.
- **Tunnel URL changes each time** → yes, `localtunnel` gives you a fresh subdomain each run. The installed app caches the first URL it saw, so it keeps working even after the tunnel URL rotates — as long as you re-run `npm run phone-app` before launching it.
- **`localtunnel` is down or slow** → as a fallback, install `cloudflared` (free, single binary) once via `winget install Cloudflare.cloudflared`, then run `cloudflared tunnel --url http://localhost:5173` in a second terminal alongside `npm run phone`.
- **Phone can't reach laptop over LAN (option B)** → firewall rules above, same Wi-Fi, router client isolation disabled.
- **App still shows browser URL bar** → you opened it from a browser tab, not the installed app. Close the tab, launch from the home screen / app drawer instead.

## How this counts as a "real app"

On modern Android, Chrome's "Install app" action creates a **WebAPK** — a real Android APK generated by Google on your behalf and signed automatically. Android treats it the same way as any Play Store app:

- Listed in app drawer and Settings → Apps
- Separate icon, separate task in the multitasking switcher
- Own storage, own permissions
- Full-screen, no browser UI
- Survives phone restarts and app updates

The only thing it doesn't do is appear in the Play Store (you didn't publish it). For your own device, that's a distinction without a difference.
